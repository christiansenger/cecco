C(++)ECCO

C++ Error Control COding is a header-only library for ECC simulations and experiments, modeling complete coding systems across arbitrary finite fields and complex inter-field relationships. CECCO provides an intuitive, object-oriented interface for finite fields, vectors, polynomials, matrices, and many linear error-correcting codes. Its low-level functionality can be used to implement arbitrary linear codes and their decoders, while its high-level functionality is convenient for creating complete simulation chains.

Source code is available on Github here.

A growing collecting of demos and examples is available on the project website.
